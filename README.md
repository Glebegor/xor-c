# Xor in c
## Description
Script to decrypt and encrypt by xor.

## Makefile
to build:

```
make build
```

to run:

```
make run
```

to build and run:

```
make build-run
```

## win1250
Binary;Hex;Unicode;Windows-1250;Character
We use ';' to separate cols in win1250.txt, so 0x3B(';' char) is hardcoded.
